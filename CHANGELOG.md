# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-21

### Added

- Initial release
- Server plugin wrapping `genericOAuth` with Transactional defaults
- Client plugin with `signIn.transactional()` convenience method
- Auto-discovery of OIDC endpoints
- PKCE support (enabled by default)
- Default profile mapper with organization claims support
- Full TypeScript support
- Comprehensive test suite
